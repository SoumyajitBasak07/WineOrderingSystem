Wine Ordering System using Java Swing

Introduction
This Wine Ordering System is a simple Java application built using Java Swing and JFrame. It allows users to order wines from a predefined selection, add them to a cart, and proceed to checkout. The system provides a user-friendly interface for browsing wines, adding them to the cart, and managing the order.

Features:
Browse Wines: Users can browse through a selection of wines available for ordering.
Add to Cart: Users can add wines to their shopping cart.
View Cart: Users can view the items they have added to their cart.
Remove from Cart: Users can remove items from their cart if needed.
Checkout: Users can proceed to checkout to finalize their order.

Requirements:
Java Development Kit (JDK)
Java IDE (Eclipse, IntelliJ IDEA, etc.)

How to Run:
Clone or download the repository to your local machine.
Open the project in your preferred Java IDE.
Compile and run the Main.java file.
The application window should appear, allowing you to interact with the Wine Ordering System.
Usage:
Upon launching the application, you will be presented with the list of available wines.
Browse through the wines and click on the ones you wish to order.
The selected wines will be added to your cart.
You can view your cart by clicking on the "View Cart" button.
In the cart, you can remove items if needed.
Once you're satisfied with your selection, proceed to checkout by clicking the "Checkout" button.
Follow the prompts to finalize your order.
